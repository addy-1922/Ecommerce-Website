# 🛒 E-Commerce Platform

A full-featured **E-Commerce Platform** built with **Python** and **Django** that provides a seamless online shopping experience. The application allows users to browse products, manage their shopping cart, place orders, and manage their accounts through a secure and user-friendly interface. An integrated Django Admin Panel enables efficient management of products, categories, users, and orders.

---

## 🚀 Features

* User registration and authentication
* Product browsing and search
* Product categories
* Product detail pages
* Shopping cart management
* Secure checkout process
* Order placement and tracking
* User profile management
* Responsive user interface
* Django Admin Panel for product and order management

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/addy-1922/ecommerce-platform.git
cd ecommerce-platform
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Apply migrations

```bash
python manage.py migrate
```

### Run the development server

```bash
python manage.py runserver
```

Open the application in your browser:

```text
http://127.0.0.1:8000/
```
